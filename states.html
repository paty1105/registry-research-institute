State Law Database

Planned state profiles will document registry requirements in every state using primary legal sources.

Each profile should track

  • Registration duration and removal options
  • In-person reporting frequency
  • Residence and exclusion-zone restrictions
  • Internet identifier reporting
  • Travel and temporary lodging reporting
  • Driver’s license or identification markings
  • Public website disclosures
  • Penalties for noncompliance
